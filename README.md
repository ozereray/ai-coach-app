# 🧠 AI Coach App

A modern AI-powered coaching application built with **React Native**, **Expo**, and **TypeScript**, designed to deliver intelligent conversations, personalized guidance, and a seamless mobile experience.

---

## ✨ Features

- 🤖 AI-powered coaching experience
- 📱 Cross-platform mobile application
- ⚡ Fast and responsive user interface
- 🎨 Clean and modern UI design
- 🔥 Built with Expo for rapid development
- 🧩 Scalable component-based architecture
- 🌐 API integration support
- 🚀 Production-ready deployment setup
- 🔐 Environment configuration support

---

## 🛠️ Tech Stack

- **Framework:** React Native
- **Platform:** Expo
- **Language:** TypeScript
- **Navigation:** React Navigation
- **Styling:** Custom UI Components
- **Deployment:** Vercel / Expo
- **Package Manager:** npm

---

## 📦 Installation

Clone the repository and install dependencies:

```bash
git clone https://github.com/ozereray/ai-coach-app.git

cd ai-coach-app

npm install
```

---

## ▶️ Running the Development Server

Start the Expo development server:

```bash
npm start
```

Run on specific platforms:

```bash
npm run android
# or
npm run ios
# or
npm run web
```

## ⚙️ Environment Setup

Create a `.env` file in the root directory:

```env
API_URL=your_api_url
OPENAI_API_KEY=your_openai_api_key
```

---

## 🚀 Deployment

Build and deploy the project using Expo or Vercel:

```bash
npm run build
```

For Expo deployment:

```bash
eas build
```
